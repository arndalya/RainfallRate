# Rainfall Rate Prediction App
Aplikasi Rainfall Rate Prediction App merupakan aplikasi berbasis web yang dibangun menggunakan Streamlit. Aplikasi ini bertujuan untuk membandingkan kinerja beberapa metode machine learning dan deep learning, yaitu Support Vector Regression (SVR), Backpropagation, dan Long Short-Term Memory (LSTM) dalam melakukan prediksi curah hujan.
Aplikasi ini memudahkan pengguna untuk melakukan eksperimen dan evaluasi model berdasarkan mode data yang dipilih (training, validasi, atau testing) dengan dataset yang telah disediakan.
